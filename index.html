Under the hood

How It Works

Three steps from download to done.

1

Watch

On launch, watchdog hooks into the Windows file-system notification API and monitors every watched folder for new file events — no polling, no CPU spin.

2

Detect

When a file with a supported archive extension appears (created or moved in from outside), a brief settle delay ensures the file is fully written before extraction begins.

3

Extract

patool delegates to the best available backend (7-Zip, unrar, etc.) and unpacks the archive into a new Extracted <name> subfolder inside Downloads.

Get started

Install in 60 Seconds

Requires Python 3.10+ and a few packages.

1 — Install dependencies

pip install watchdog pystray pillow patool

2 — Run the app

python unzipadeedownloads.py

3 — Or build a standalone exe

pip install pyinstaller
pyinstaller --onefile --noconsole --add-data "icon.png;." --icon="icon.png" unzipadeedownloads.py

For .rar and .7z support, install 7-Zip and ensure 7z.exe is on your PATH.
